# YE CARS

Premium occasion website built with Next.js 16 and React 19.

## Pages

- Home
- Aanbod with client-side filters and sorting
- Static vehicle detail pages
- Diensten
- Over ons
- Verkocht
- Contact

## Development

```bash
npm install
npm run dev
```

Production verification:

```bash
npm run lint
npm run build
```

## Content

Company contact details are centralized in `src/data/site.ts`.
Inventory and vehicle specifications are centralized in `src/data/cars.ts`.

The current phone number and address are placeholders and must be replaced with
the final YE CARS business details before launch.
